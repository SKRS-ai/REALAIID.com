Authentication

All requests to the Commonwealth API must be authenticated with a Bearer Token. Enterprise partners can generate these tokens via the **Partnership Vault**.

Authorization: Bearer COMMONWEALTH_SECRET_KEY

Verify Human Status

Use this endpoint to confirm if a specific Likeness Hash corresponds to a certified "Human Anchor" in the Commonwealth.

POST/v1/anchor/verify

Request Body

{ "likeness_hash": "a1b2c3d4e5f6...", "session_token": "pulse_session_7721", "vssn_required": true }

Response

{ "status": "CERTIFIED_HUMAN", "residency_suite": "7721-PHL", "verification_timestamp": "2026-03-27T15:22:02Z", "anchor_node": "Philadelphia_01" }
